Connexion.me

Connexion.me is a free and easy tool to organize your bookmarks. It is open source, meaning that you can download and install the code on your own server. The goal is to make Connexion.me more context aware, meaning that it will be able to pull meta information from websites and create connections with other bookmarks. A knowledge discovery tool...
